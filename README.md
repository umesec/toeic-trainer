# 📘 TOEIC トレーナー

TOEIC 対策のための英語学習アプリ。**Expo / React Native**（TypeScript）製で、開発中はブラウザで動かしつつ、同じコードをそのまま iOS アプリとして展開できます。

## 起動方法

```bash
npm install
npx expo start
```

- **ブラウザで確認**: 起動後にターミナルで `w` を押す（http://localhost:8081）
- **iPhone 実機で確認**: App Store から [Expo Go](https://expo.dev/go) を入れ、ターミナルの QR コードを読み取る（PC と同一 Wi-Fi 必須）
- **App Store 配布**: 将来的には [EAS Build](https://docs.expo.dev/build/introduction/) でビルド

## 機能

### 🎯 学習プラン（ホーム）
目標スコア・現在スコア・試験日を設定すると、学習プログラムを自動生成します。

- 期間を **語彙固め期 → 演習期 → 直前総仕上げ期** の3フェーズに分割
- 目標との差分と残り日数から学習強度を算出し、**「今日のメニュー」**（単語◯枚・クイズ◯問・リスニング◯回、目安時間つき）を毎日提示
- 各タブでの学習が自動でメニューの消化状況に反映される

### 📚 単語フラッシュカード
- TOEIC 頻出単語 **150語** 収録（品詞・訳・例文・例文訳つき、音声再生対応）
- **間隔反復（SRS、SM-2簡易版）**: 「もう一度／難しい／普通／簡単」の自己評価で次回出題日が決まる
- 自分の単語を追加できる単語帳機能

### 📖 TOEIC ガイド（初学者向け）
- 試験の全体像（2時間200問・スコア10〜990点）とスコアの仕組みを解説
- Part 1〜7 それぞれの形式・問題数・攻略ポイントを紹介し、アプリ内の対応する練習画面へ直接ジャンプできる
- 学習の進め方と受験当日のTips（先読み・時間配分・塗り絵）も収録

### 📝 文法・語彙クイズ
- TOEIC Part 5 形式の4択問題 **80問**（品詞／時制／前置詞／語彙）
- 回答直後に解説と訳を表示
- 間違えた問題は記録され、次回セッションで優先的に再出題

### 🗣️ Part 3/4 会話・トーク
- 会話（Part 3）×5セット + トーク（Part 4）×5セット、各3問の本番形式（計30問）
- 会話は話者ごとに声の高さを変えて逐次読み上げ。答え合わせでスクリプト全文と訳を確認できる

### 📄 Part 6/7 長文問題
- Part 6 長文穴埋め: 5文書×4空所（文挿入問題を含む）
- Part 7 長文読解: メール／広告／告知／記事／チャット／ダブルパッセージの6文書×3問（NOT問題・意図問題・言い換え問題を含む）
- 全問回答後に全文訳を表示できる

### 🎧 リスニング：音声変化（コア機能）
「知っている単語なのに聞き取れない」原因になる英語特有の音声変化を、実例と音声つきで学べます。

- 収録ルール: **フラップT／脱落（Elision）／連結（Linking）／同化（Assimilation）／弱形**
- 各ルールに実例フレーズ 8〜9 個。「文字通りの発音」と「実際の音」をカナで対比表示し、変化箇所をハイライト
- 🔊 通常速度／🐢 ゆっくり の2段階再生
- **ディクテーション練習**: 音声変化を含む文を聞いて書き取り、答え合わせで変化箇所の解説を表示

### 🎙️ Part 2 応答問題
- 本番の Part 2 形式 **35問**。質問と応答 (A)(B)(C) を音声のみで聞き、最も適切な応答を選ぶ
- 回答後に質問文・訳・「引っかけの型」の解説を表示（Where→場所、選択疑問文にYes/Noは不可、など）

### ⏱ 模試・実力測定（推定スコア）
- **ミニ模試**: 10分・15問（Part 2×5 + Part 5×10）。スキマ時間の腕試し用
- **実力測定モード**: 40分・65問（L: Part 2×13 + Part 3/4×5セット ／ R: Part 5×20 + Part 6×2文書 + Part 7×3文書）
- 実力測定モードでは終了後に **推定スコア（10〜990点、±マージンつき）** を表示。正答率→スコアの区分線形換算（公式換算表の近似）+ 二項分布の標準誤差ベースの不確かさ表示
- 本番同様、回答中は正誤が表示されない。結果画面でセクション別成績・誤答復習リスト・受験履歴（推定スコア含む）を確認できる

### 📊 弱点分析
- 分野（品詞／時制／前置詞／語彙）ごとの累計正答率を自動記録
- クイズ画面に一覧表示し、最も苦手な分野を提示。分野を絞った集中練習につなげられる

### 進捗管理
- 連続学習日数（ストリーク）、クイズ累計正答率、復習待ちカード数
- すべてローカル保存（AsyncStorage）。アカウント不要・オフライン動作

## 構成

```
src/
├── app/               # 画面（expo-router）
│   ├── index.tsx      # ホーム: 学習プラン・今日のメニュー・進捗
│   ├── flashcards.tsx # 単語カード（SRS）
│   ├── quiz.tsx       # Part 5 形式クイズ
│   └── listening/     # 音声変化カタログ・詳細・ディクテーション
├── data/              # 学習データ（単語・クイズ・音声変化）
├── lib/               # ロジック（srs / plan / storage / speech）
└── components/        # UI部品
```

## 制約

- 音声は TTS（OS の読み上げ）のため、音声変化の再現度は OS 音声品質に依存します。データ構造に `audioUrl` を持たせてあり、将来ネイティブ録音音声へ差し替え可能です
- 学習プランの「今日のメニュー」は目安であり、スコア保証をするものではありません
- 実力測定モードの推定スコアは65問（本番は200問）による簡易推定で、**±40〜60点程度のブレがある目安**です。TTS音声・自作問題である点も含め、公式スコアを保証するものではありません
